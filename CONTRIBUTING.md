# Contributing

Thanks for your interest in contributing to ghostty-claude-theme!

## How to contribute

1. Fork the repo
2. Create a branch (`git checkout -b my-change`)
3. Make your changes
4. Test the install script locally: `bash install.sh`
5. Open a pull request

## What to contribute

- Color palette tweaks (keep the warm, earthy aesthetic)
- Config improvements (must be valid [Ghostty options](https://ghostty.org/docs/config/reference))
- Bug fixes in the install/uninstall scripts
- Documentation improvements

## Guidelines

- Keep it simple. This is a theme, not a framework.
- Test on macOS before submitting. The install script uses BSD `sed` and `grep`.
- If you change colors, explain why in your PR description.

## Questions?

Open an issue. No question is too small.
